# Calculadora de IMC

Projeto de uma Calculadora de IMC (Índice de Massa Corporal) desenvolvida com HTML, CSS e JavaScript puro, permitindo ao usuário calcular seu IMC e visualizar a classificação correspondente.

## Funcionalidades

✅ Cálculo automático do IMC

✅ Validação de entrada (aceita apenas números e vírgula)

✅ Conversão automática de vírgula para ponto

✅ Exibição dinâmica da classificação

✅ Tabela com todas as faixas de IMC

✅ Alteração visual conforme o resultado

✅ Botão para limpar dados

✅ Alternância entre tela de cálculo e tela de resultado

## Como funciona o cálculo

O IMC é calculado utilizando a fórmula:

IMC = peso ÷ (altura × altura)

Exemplo:
Peso: 70 kg
Altura: 1,70 m

IMC = 70 ÷ (1.70 × 1.70) = 24.2

### Como baixar e rodar o projeto
- Opção 1 — Baixando pelo GitHub (sem Git instalado)

Clique no botão verde Code

Clique em Download ZIP

Extraia o arquivo ZIP

Abra a pasta do projeto

Clique duas vezes no arquivo index.html

O projeto abrirá diretamente no navegador.

- Opção 2 — Clonando com Git

Copie a URL do repositório

Abra o terminal na pasta onde deseja salvar o projeto

Execute o comando de clonagem usando a URL do repositório

Entre na pasta criada

Abra o arquivo index.html no navegador
